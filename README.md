# Website Testing Using Selenium IDE (HubSpot)

## Project Overview

This project demonstrates **automated functional testing of the HubSpot website using Selenium IDE**. The objective is to verify whether important website functionalities are working correctly by creating and executing structured test cases.

The testing process compares **expected results with actual outputs** to validate the correct behavior of the application.

---

## Project Description

In this project, multiple test cases were designed and executed to test key features of the HubSpot website. The testing focuses on validating different user interactions such as opening chat support, requesting a demo, selecting pricing plans, browsing products, and accessing resources.

Each test case includes:

* Test Case ID
* Test Scenario
* Pre-conditions
* Test Steps
* Test Data
* Expected Output
* Actual Output
* Status of the test

All test cases were executed successfully and the results were documented.
